# Theater
Two apps
- Rails REST API
- Admin panel

Установка и запуск
```
git clone https://github.com/vovanmozg/theater-tp.git
cd theater-tp
docker-compose build
docker-compose up -d
```

## Admin panel
http://localhost:3000

## API 
http://localhost:3001
